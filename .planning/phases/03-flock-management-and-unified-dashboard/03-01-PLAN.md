    5. Create `async def periodic_flock_loop(db_pool)`: runs every 30 minutes (per D-02). Each cycle:
       - Query the latest `nesting_box_weight` reading from sensor_readings where zone_id='coop' and quality='GOOD' and time > now() - 35 minutes.
       - If a reading exists, call `estimate_egg_count(reading_value, flock_config)`.
       - UPSERT into egg_counts for today's date.
       - Send `nesting_box` delta via `notify_api()` with fields: type="nesting_box", estimated_count, hen_present, raw_weight_grams, updated_at.
       - Evaluate production drop alert: query last 3 days of egg_counts. If < 3 days exist, skip. Otherwise compute 3-day rolling average, compare to expected (from compute_expected_production using flock_config). Call `alert_engine.evaluate("production_drop:coop", ratio_pct, 75.0, clear_above=True)` where ratio_pct = (rolling_avg / expected) * 100. If alert state changed, send alert_state delta.
       - **Pitfall 4 guard:** If fewer than 3 egg_count records exist, skip alert evaluation entirely.

    6. Create `async def daily_feed_loop(db_pool)`: runs once daily at midnight UTC. Each cycle:
       - Query yesterday's first and last `feed_weight` readings from sensor_readings where zone_id='coop' and quality='GOOD'.
       - Call `compute_daily_feed_consumption(start_weight, end_weight)`.
       - INSERT into feed_daily_consumption.
       - Query last 7 days of feed_daily_consumption (exclude refill_detected=True and NULL consumption days).
       - Compute 7-day rolling average. Compute today's deviation from average.
       - Send `feed_consumption` delta via `notify_api()` with fields: type="feed_consumption", rate_grams_per_day (today's or latest available), weekly (7 values oldest-first, nulls as 0).
       - Evaluate feed_consumption_drop alert: if today's consumption is >30% below 7-day average, fire. Call `alert_engine.evaluate("feed_consumption_drop:coop", deviation_pct, 30.0, clear_above=True)`.
       - **Pitfall 4 guard:** If fewer than 3 days of consumption data exist, skip alert evaluation.
       - Sleep until next midnight UTC.

    7. In `_evaluate_phase2()` (or create new `_evaluate_phase3()`), add handling for sensor_type == "nesting_box_weight": this is the per-reading path. On each reading, compute egg estimate and hen_present, then send nesting_box delta immediately. Also UPSERT today's egg_counts. This provides real-time updates in addition to the 30-minute periodic loop.
