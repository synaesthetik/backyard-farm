<objective>
Restructure the SvelteKit routes for a 4-tab layout and build the Home tab as the unified
overview screen (UI-01). This moves the Zones list from / to /zones, adds a new Home tab at /
with compact zone cards and a flock summary card, extends the WebSocket store with Phase 3
flock state fields, and adds Phase 3 TypeScript types.

Purpose: The Home tab is the farmer's single-screen command center. Everything visible at a
glance: zone health across all garden beds plus flock status. This is the most impactful UX
change in Phase 3.

Output: New Home tab, updated TabBar, moved Zones route, FlockSummaryCard, ZoneCard compact
variant, extended WS store and TS types.
</objective>
