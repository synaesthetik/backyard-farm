<threat_model>
## Threat Model — Hub Service Stack

**Assets:** TimescaleDB sensor data, MQTT broker credentials, hub network services.

**Trust boundary:** LAN only. No cloud, no internet-facing services.

**Threats (STRIDE):**

| Threat | Category | Severity | Mitigation |
|--------|----------|----------|------------|
| Unauthorized MQTT access | Spoofing | HIGH | Mosquitto requires explicit auth (allow_anonymous false); password file generated with mosquitto_passwd. Addressed in plan 01-02. |
| TimescaleDB exposed on network | Information Disclosure | MEDIUM | Bind TimescaleDB to localhost only (127.0.0.1:5432) in docker-compose; no external port mapping. |
| Caddy serving to internet | Information Disclosure | LOW | Caddy binds to LAN interface only; router firewall blocks external 443. User's LAN-only constraint. |
| Docker socket exposure | Elevation of Privilege | LOW | No Docker socket mounted into containers. Standard practice. |
| Unencrypted MQTT | Tampering | LOW | LAN-only; MQTT TLS deferred (standard for local IoT). ACL + password is sufficient for Phase 1 LAN. |

**HIGH severity mitigations:** Mosquitto auth is mandatory and addressed in plan 01-02. TimescaleDB localhost binding is implemented in this plan.
</threat_model>

**hub/init-db.sql:**
```sql
CREATE EXTENSION IF NOT EXISTS timescaledb;

CREATE TABLE IF NOT EXISTS sensor_readings (
  time        TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  zone_id     TEXT NOT NULL,
  sensor_type TEXT NOT NULL,
  value       DOUBLE PRECISION NOT NULL,
  quality     TEXT NOT NULL CHECK (quality IN ('GOOD', 'SUSPECT', 'BAD')),
  stuck       BOOLEAN NOT NULL DEFAULT FALSE,
  raw_value   DOUBLE PRECISION,
  calibration_applied BOOLEAN NOT NULL DEFAULT FALSE,
  received_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

SELECT create_hypertable('sensor_readings', 'time');

CREATE INDEX idx_sensor_readings_zone ON sensor_readings (zone_id, time DESC);
CREATE INDEX idx_sensor_readings_quality ON sensor_readings (quality, time DESC);

CREATE TABLE IF NOT EXISTS calibration_offsets (
  zone_id     TEXT NOT NULL,
  sensor_type TEXT NOT NULL,
  offset_value DOUBLE PRECISION NOT NULL DEFAULT 0.0,
  dry_value   DOUBLE PRECISION,
  wet_value   DOUBLE PRECISION,
  temp_coefficient DOUBLE PRECISION DEFAULT 0.0,
  updated_at  TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  PRIMARY KEY (zone_id, sensor_type)
);

CREATE TABLE IF NOT EXISTS zone_config (
  zone_id       TEXT PRIMARY KEY,
  name          TEXT NOT NULL,
  plant_type    TEXT,
  soil_type     TEXT,
  target_vwc_min DOUBLE PRECISION,
  target_vwc_max DOUBLE PRECISION,
  target_ph_min  DOUBLE PRECISION,
  target_ph_max  DOUBLE PRECISION,
  irrigation_zone_id TEXT,
  created_at    TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  updated_at    TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE TABLE IF NOT EXISTS node_heartbeats (
  node_id     TEXT NOT NULL,
  heartbeat_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  PRIMARY KEY (node_id, heartbeat_at)
);

SELECT create_hypertable('node_heartbeats', 'heartbeat_at');
```

**hub/dashboard/server.js** — Custom Node server per Research Pattern 3:
```javascript
import { createServer } from 'http';
import { WebSocketServer } from 'ws';
import { handler } from './build/handler.js';

const PORT = process.env.PORT || 3000;

const server = createServer(handler);
const wss = new WebSocketServer({ noServer: true });

server.on('upgrade', (req, socket, head) => {
  if (req.url === '/ws/dashboard') {
    wss.handleUpgrade(req, socket, head, (ws) => {
      wss.emit('connection', ws, req);
    });
  } else {
    socket.destroy();
  }
});

wss.on('connection', (ws) => {
  console.log('Dashboard WebSocket client connected');
  ws.send(JSON.stringify({ type: 'snapshot', zones: {}, nodes: {} }));
  ws.on('close', () => console.log('Dashboard WebSocket client disconnected'));
});

server.listen(PORT, () => {
  console.log(`Dashboard server listening on port ${PORT}`);
});
```
