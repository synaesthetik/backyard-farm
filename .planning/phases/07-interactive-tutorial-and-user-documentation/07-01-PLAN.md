**Tutorial layout** (`hub/dashboard/src/routes/tutorial/+layout.svelte`):
```svelte
<script lang="ts">
  import { onMount } from 'svelte';
  import { page } from '$app/stores';

  const TOTAL_STEPS = 8;
  const STORAGE_KEY = 'tutorial_step';

  let { children } = $props();
  let currentStep = $state(1);

  onMount(() => {
    const saved = localStorage.getItem(STORAGE_KEY);
    if (saved) currentStep = parseInt(saved, 10);
  });

  const stepFromUrl = $derived(() => {
    const match = $page.url.pathname.match(/\/tutorial\/(\d+)/);
    return match ? parseInt(match[1], 10) : currentStep;
  });
</script>

<div class="tutorial-shell">
  <div class="tutorial-progress-bar" role="progressbar"
       aria-valuenow={stepFromUrl()} aria-valuemin={1} aria-valuemax={TOTAL_STEPS}>
    <div class="tutorial-progress-fill"
         style="width: {(stepFromUrl() / TOTAL_STEPS) * 100}%"></div>
  </div>
  <div class="tutorial-step-counter">
    Step {stepFromUrl()} of {TOTAL_STEPS}
  </div>
  {@render children()}
</div>
```

**Shared step helper** — each step page (`1/+page.svelte` through `8/+page.svelte`) follows this pattern:
```svelte
<script lang="ts">
  import { goto } from '$app/navigation';
  import { CheckCircle, ChevronLeft, ChevronRight } from 'lucide-svelte';

  const STEP = N;           // replace N with step number
  const TOTAL_STEPS = 8;
  const COMPLETE_KEY = 'tutorial_completed';
  const STORAGE_KEY = 'tutorial_step';

  function markDone() {
    if (STEP >= TOTAL_STEPS) {
      localStorage.setItem(COMPLETE_KEY, 'true');
      localStorage.removeItem(STORAGE_KEY);
      goto('/tutorial/8');  // stay on step 8 but mark complete
    } else {
      const next = STEP + 1;
      localStorage.setItem(STORAGE_KEY, String(next));
      goto(`/tutorial/${next}`);
    }
  }
</script>

<svelte:head><title>Farm Tutorial — Step N: [Title]</title></svelte:head>

<div class="step-card">
  <h2 class="step-heading">[Step title]</h2>
  <div class="step-body">[Step instructions]</div>
  <div class="step-actions">
    {#if STEP > 1}
      <a href="/tutorial/{STEP - 1}" class="btn-secondary">
        <ChevronLeft size={16} /> Back
      </a>
    {/if}
    <button class="btn-primary" onclick={markDone}>
      {#if STEP >= TOTAL_STEPS}
        Finish <CheckCircle size={16} />
      {:else}
        I did this <ChevronRight size={16} />
      {/if}
    </button>
  </div>
</div>
```

```
Title: Verify Sensor Data

With your garden node powered and connected to the same Wi-Fi/LAN as the hub:

2. Check the quality badge: GOOD means the reading passed calibration checks
3. If the badge shows SUSPECT or BAD, see the troubleshooting guide
4. If the reading shows STALE (>5 min old), check that the node is online in the System Health panel

Action link: <a href="/">Open Dashboard Home →</a>
```

```
Title: Run a Manual Irrigation
Body:
Open a Zone detail page and locate the Irrigation section. Click "Open Valve" to start irrigation for that zone. The hub enforces a single-zone rule — only one valve can be open at a time. Watch the valve status update to "open" within 5 seconds. Click "Close Valve" to stop. Confirm the dashboard shows the zone irrigation status return to "closed".
```

```
Title: Set Up Coop Automation

Save settings. The coop door will open and close automatically each day. You can always use the manual open/close override on the Coop tab.

Action link: <a href="/coop/settings">Open Coop Settings →</a>
```
